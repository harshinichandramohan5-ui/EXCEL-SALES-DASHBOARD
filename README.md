# 📊 Excel Sales Dashboard

## 📌 Project Overview
This project presents an interactive Sales Dashboard developed in Microsoft Excel. It transforms raw sales data into meaningful business insights using Pivot Tables, Pivot Charts, Slicers, Conditional Formatting, and Excel formulas. The dashboard enables users to monitor key performance indicators (KPIs), analyze sales trends, and evaluate business performance through an intuitive interface.

---

## 🎯 Objectives
- Analyze overall sales performance.
- Monitor revenue and profit trends.
- Compare sales across different regions and product categories.
- Identify top-performing products and customers.
- Enable interactive filtering for better decision-making.

## ⚙️ Configuration
1. Download the project files.
2. Open the Excel workbook using Microsoft Excel (2019 or Microsoft 365 recommended).
3. Enable editing if prompted.
4. Refresh Pivot Tables using **Data → Refresh All**.
5. Use the slicers to interact with the dashboard.
6. If using a new dataset, ensure the column names remain consistent before refreshing.
